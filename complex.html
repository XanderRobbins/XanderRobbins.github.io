Module 01

Propositional Logic

What is a Proposition?

A proposition is a declarative sentence that is either true or false, but not both.

Interrogative and imperative sentences like "Did you try binary search?" or "Always comment your code" are NOT propositions — they can't be true or false.
✅ Proposition

"Washington D.C. is the capital of the USA." — Has a definite truth value (True).

✅ Proposition

"33 is a prime number." — Has a definite truth value (False: 33 = 3×11).

✅ Proposition

"Every even integer > 2 is the sum of two primes." — Goldbach's Conjecture, unknown but valid proposition.

❌ Not a Proposition

"What time is it?" — Interrogative, cannot be true or false.

❌ Not a Proposition

"x + 1 = 3" — Contains a free variable; truth depends on x.

Conditional Statements — Multiple Forms

The conditional statement p → q can be expressed many ways in English:

"if p, then q"
"p implies q"
"p only if q"
"q if p"
"q whenever p"
"q is necessary for p"
"p is sufficient for q"
"q unless ¬p"

Converse, Inverse, Contrapositive

From an implication p → q, we can derive three related statements:

🔄 Interactive Builder — enter p and q in plain English

Original (p → q)
If it is raining, then I need an umbrella.
Converse (q → p)
If I need an umbrella, then it is raining.
Inverse (¬p → ¬q)
If it is not raining, then I do not need an umbrella.
Contrapositive (¬q → ¬p)
If I do not need an umbrella, then it is not raining.
💡 The original and contrapositive are logically equivalent. The converse and inverse are logically equivalent to each other.

Module 02

Propositional Equivalencies

Tautologies, Contradictions, and Contingencies

✅ Tautology

Always true regardless of truth assignments.

p ∨ ¬p ≡ T
❌ Contradiction

Always false. Also called unsatisfiable.

p ∧ ¬p ≡ F
⚡ Contingency

Neither — sometimes true, sometimes false.

p → q
A proposition is satisfiable if true under at least one assignment. Contradictions are not satisfiable.

Build a circuit that computes (A ∧ B) ∨ (¬A ∧ C) using the builder above. What is its output when A=1, B=0, C=1?
When A=1, B=0, C=1: (1∧0)∨(¬1∧1) = 0∨(0∧1) = 0∨0 = 0. Wait — ¬A=0, so ¬A∧C = 0∧1 = 0. And A∧B = 1∧0 = 0. So output = 0.

Module 04

Predicates & Quantifiers

When Propositional Logic Fails

Propositional logic can't capture arguments like:

"All candy made with chocolate is delicious."
"M&M's are made with chocolate."
∴ M&M's are delicious.

We need predicate logic — which adds variables, predicates, and quantifiers.

Variables

x, y, z — the subjects of our statements.

Predicates

Properties variables can have. P(x): "x is greater than 3".

Domain (Universe U)

The set of all possible values for variables. Must be specified!

Quantifiers

∀

Universal Quantifier — "For All"

∀x P(x) — P(x) is true for every x in the domain.

True when: P(x) holds for every x in the domain
False when: There exists at least one counterexample
∃

Existential Quantifier — "There Exists"

∃x P(x) — P(x) is true for at least one x.

True when: At least one x in the domain satisfies P(x)
False when: P(x) is false for every x in the domain

Quantifier Practice

Let P(x) = "x² > 0" with domain ℤ. Is ∀x P(x) true or false? If false, give a counterexample value of x.
False — P(0) = "0² > 0" = "0 > 0" = False. x = 0 is a counterexample!
Let P(x) = "x² > 0" with domain ℤ. Is ∃x P(x) true or false?
True — P(1) = "1 > 0" = True.
Negate: "There is an honest politician." Write the negation in English (start with "No" or "Every").
Original: ∃x H(x)
Negation: ¬∃x H(x) ≡ ∀x ¬H(x)
"No politician is honest."
Negate "All Americans eat cheeseburgers" using quantifier notation. What symbol starts your answer?
Original: ∀x C(x)
Negation: ¬∀x C(x) ≡ ∃x ¬C(x)
"There is some American who does not eat cheeseburgers."

Translation Practice

Some student in this class has visited Mexico.

Module 05

Nested Quantifiers

Order Matters!

"Every real number has an additive inverse."
∀x ∃y (x + y = 0)
∀x ∀y P(x,y)

For all x and y, P(x,y) is true. Symmetric.

∃x ∃y P(x,y)

There exist some x and y where P is true. Symmetric.

∀x ∃y P(x,y)

For every x, there exists some y (can differ per x).

∃y ∀x P(x,y)

One fixed y that works for ALL x. Much stronger!

⚠️ ∀x ∃y P(x,y) and ∃y ∀x P(x,y) are usually NOT equivalent!

Negating Nested Quantifiers

¬∀x ∃y P(x,y)
≡
∃x ∀y ¬P(x,y)
Flip each quantifier, negate the predicate
¬∃x ∀y P(x,y)
≡
∀x ∃y ¬P(x,y)
Flip each quantifier, negate the predicate

Practice Problems

Let Q(x,y) = "x + y = 5" over ℝ. Is ∀x ∃y Q(x,y) true or false?
True — For any x, choose y = 5 − x. Each x gets its own y.
Is ∃y ∀x Q(x,y) true (Q(x,y) = "x + y = 5")? Type true or false.
False — No single y makes x + y = 5 for ALL x simultaneously.
Let P(x,y) = "x·y = 0" over ℝ. Is ∃x ∀y P(x,y) true? (Hint: try x = 0)
∀x∀y: False — 2·3 ≠ 0
∀x∃y: True — choose y = 0
∃x∀y: True — choose x = 0
∃x∃y: True — x = 0, y = 0

Proof Practice

Step-Through Proof Problems

From p ∧ (p → q), what rule do you apply to get q? (Name the rule of inference used in the final step)
Step 1: p ∧ (p → q)    [Premise]
Step 2: p    [Simplification]
Step 3: p → q    [Simplification]
Step 4: ∴ q    [Modus Ponens, 2, 3] ✓
To go from "∀x (D(x) → L(x))" and "c = Oliver" to "D(Oliver) → L(Oliver)", which rule is used?
Step 1: ∀x (D(x) → L(x))    [Premise]
Step 2: D(Oliver)    [Premise]
Step 3: D(Oliver) → L(Oliver)    [Universal Instantiation, c = Oliver]
Step 4: ∴ L(Oliver)    [Modus Ponens, 2, 3] ✓
In the proof: Premises (p ∧ t)→(r ∨ s), q→(u ∧ t), u→p, ¬s, q. What is the final conclusion?
1: q [Premise] → 2: u ∧ t [MP] → 3: u, t [Simp] → 4: p [MP] → 5: p ∧ t [Conj] → 6: r ∨ s [MP] → 7: ∴ r [Disj. Syllogism with ¬s] ✓
